# 🖼️ คู่มือสัดส่วนรูปภาพประจำระบบ (Aspect Ratio & Image Inventory)

เพื่อควบคุมไม่ให้เกิดบั๊กรูปภาพบิดเบี้ยว ยืดผิดสัดส่วน หรือเกิดระยะกระตุกระหว่างดาวน์โหลดรูปภาพ (Layout Shift) หน้าคู่มือนี้ทำการรวบรวม **สัดส่วนภาพจริง (Canonical Aspect Ratios)** ที่ใช้อยู่ในโครงสร้างปัจจุบันทั้งหมด เพื่อให้ทีมดีไซเนอร์สอดรับเลย์เอาต์การ์ดและเฟรมบน Figma ได้อย่างแม่นยำ

---

## 📐 1. สรุปประเภทสัดส่วนภาพในระบบ (Aspect Ratio Index)

ระบบปัจจุบันใช้งานสัดส่วนภาพหลัก ๆ อยู่ 4 รูปแบบในการจัดองค์ประกอบหน้าเว็บ:

| สัดส่วนภาพ (Aspect Ratio) | Tailwind Class | พิกเซลแนะนำ (พอร์ตเทรต/แลนด์สเคป) | ตัวอย่างการใช้งานจริงในหน้าเว็บ |
| :--- | :--- | :--- | :--- |
| **`2:3` (Editorial Portrait)** | `aspect-[2/3]` | `400 x 600 px` | การ์ดผลงานซีรีส์ / อัลบั้มเพลงหลัก (Content Section) |
| **`3:4` (Fashion portrait)** | `aspect-[3/4]` | `600 x 800 px` | ภาพถ่ายแฟชั่น / การ์ดโปรไฟล์ / ภาพไฮไลต์โมบาย |
| **`16:8` / `16:9` (Wide Screen)** | `aspect-[16/8]` | `1200 x 600 px` | ภาพสไลด์และแบนเนอร์หลักเดสก์ท็อป (Hero Banner / Slides) |
| **`9:16` (Vertical Mobile)** | `aspect-[9/16]` | `1080 x 1920 px` | ภาพสไลด์แบนเนอร์สำหรับจอมือถือ (Hero Slides Mobile) |
| **`1:1` (Square)** | `aspect-square` | `500 x 500 px` | โลโก้แบรนด์สินค้า Collab / ภาพอวาตาร์แฟนคลับ |

## 🖼️ 2. เจาะลึกแต่ละคอมโพเนนต์และการตั้งค่า (Image Component Spec)

### 🎬 2.1 การ์ดผลงานหลัก (Content Section Card)
*   **สัดส่วน:** `aspect-[2/3]` (Editorial Portrait)
*   **โค้ดเดิม:** [ContentCard.tsx](file:///d:/_DEV/namtanfilm-website/components/content/ContentCard.tsx#L97)
*   **สไตล์ดีไซน์:** รูปทรงการ์ดพอร์เทรตขอบมนกว้างพิเศษโค้งมนสูง (`rounded-[2rem]`) ล้อมด้วยเส้นขอบบาง เมื่อผู้ใช้เอาเมาส์มาวางจะเกิดการขยาย Blur ขนาดภาพ และมีปุ่มลิงก์สตรีมมิ่งโผล่ขึ้นมา

### 📸 2.2 อาร์ตเวิร์กแฟชั่นและโปรไฟล์ (Fashion & Profile Widgets)
*   **สัดส่วน:** `aspect-[3/4]` (Fashion Portrait)
*   **โค้ดเดิม:** `ProfileSection.tsx` (ภาพถ่ายดาราหลัก), `FashionSection.tsx` (ภาพงานแฟชั่นแบรนด์)
*   **สไตล์ดีไซน์:** อิงตามสัดส่วนภาพถ่ายแฟชั่นมาตรฐานแมกกาซีน ช่วยแสดงรูปภาพคนแบบเต็มตัวหรือครึ่งตัวได้สง่างามและมีขอบคมชัด

### 🖼️ 2.3 แบนเนอร์หัวเว็บสไลด์ (Hero Banner / Slides Preview)
*   **สัดส่วน:** `aspect-[16/8]` (สำหรับ Desktop) และ `aspect-[9/16]` (สำหรับ Mobile)
*   **โค้ดเดิม:** [HeroPreviewPanel.tsx](file:///d:/_DEV/namtanfilm-website/components/admin/hero/HeroPreviewPanel.tsx#L19)
*   **สไตล์ดีไซน์:** โหลดเต็มความกว้าง Above-the-fold เพื่อความ Premium หรูหราทันทีที่เข้าเว็บ

## 💡 คำแนะนำสำหรับการออกแบบ Figma
1.  **ดีไซเนอร์กรุณาวาดหน้ากากครอบภาพ (Masking frame)** ให้ล็อกตาม Aspect Ratio 5 ตัวด้านบนนี้เสมอ
2.  **หลีกเลี่ยงการใช้ Aspect Ratio แปลกๆ** (เช่น 5:7 หรือ 11:14) โดยไม่มีความจำเป็น เพื่อป้องกันการเขียน CSS พิเศษที่อาจกวนระบบ responsive ในเครื่องอื่น
3.  **การมนขอบรูปภาพ** แนะนำให้ใช้ตามมาตรฐาน `rounded-xl` (16px) หรือ `rounded-2xl` (24px) เสมอเพื่อให้อารมณ์พรีเมียมไร้รอยต่อ
